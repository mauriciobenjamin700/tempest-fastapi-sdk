UI language. To avoid duplicate symbol anchors, the rendered reference is

[**Open the API Reference »**](../reference/)

!!! tip "Searching"
    Every public symbol exported by `tempest_fastapi_sdk` is documented there
    with its full signature, parameters, return type, raised exceptions and
    source location. Use the search bar (or press `/`) to jump to a symbol by
    name — the full-text index includes the docstrings.
